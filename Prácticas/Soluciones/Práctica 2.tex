    \item Error de utilizar el cuantificador $\forall$ con $\wedge$, que hace siempre falso el predicado. Para solucionarlo se debe cambiar el $\yLuego$ por $\implicaLuego$
    \item Error de utlizar el cuantificador $\exists$ con $\rightarrow$. Para solucionarlo hay que cambiar el $\implicaLuego$ por $\yLuego$
\end{enumerate}
